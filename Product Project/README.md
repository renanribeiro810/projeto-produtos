### BASE DE DADOS
- A aplicação está configurada para usar o banco MySQL 5.1 utilzando o esquema **eribox_test** para acesso localhost.
- Na pasta database há um script de criação da tabela com os dados básicos. (podem ser criados outros)

### INSTRUÇÕES GERAIS
- Criar um CRUD de 'Produto' usando as classes, páginas e configurações já existentes na aquitetura do projeto.
- O código do produto deve ser único.
- O preço do produto deve ser maior que zero.
- O produto deve ter ao menos informaçõess de nome, código e preço.
- Todos os campos são obrigatórios.
- Deve-se permitir pesquisar produtos por Nome e/ou Código. Caso não informado um destes atributos, trazer todos.
- A arquitetura do projeto foi desenvolvida utilizando JSF (Primefaces), Spring Framework e JPA (Hibernate provider) e Tomcat 9.
- A estutura base que deverá ser usada para a construção do projeto está no gihub no repositório https://github.com/jonasvm/Eribox-Test
- Caso sinta necessidade, novas classes, atributos, configurações e páginas podem ser adicionadas, porém as já¡ existentes devem ser utilizadas.
- Validações, coesão, acoplamento, estética e qualidade do código serão levados em consideração na avaliação.
